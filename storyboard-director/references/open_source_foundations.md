# Open-Source Foundations

Use this file to keep the skill grounded in verified professional-team and research-team projects.

## Primary foundations

### ViMax

Use for:

- script-to-shot orchestration
- shot-level storyboard design
- consistency validation
- automated reference selection

Role in this skill:

- overall storyboard pipeline shape
- shot-first planning

URL:

- https://github.com/HKUDS/ViMax

### FilmAgent

Use for:

- director/screenwriter/cinematographer role decomposition
- critique-correct-verify loop
- camera-setting revision

Role in this skill:

- internal review workflow
- fail-first correction logic

URL:

- https://github.com/HITsz-TMG/FilmAgent

### MovieAgent

Use for:

- hierarchical multi-scene planning
- camera settings and cinematography structure
- long-form scene decomposition

Role in this skill:

- scene objective -> shot sequence planning
- top-down beat organization

URL:

- https://github.com/showlab/MovieAgent

### Story2Board

Use for:

- panel-level prompt shaping
- story text to storyboard prompt conversion
- board diversity with narrative continuity

Role in this skill:

- final storyboard prompt shape
- panel-minded shot phrasing

URL:

- https://github.com/DavidDinkevich/Story2Board

### TaleCrafter

Use for:

- story-to-layout thinking
- multi-character consistency
- text-to-layout control

Role in this skill:

- multi-character blocking and readable layout

URL:

- https://github.com/AILab-CVC/TaleCrafter

### Storyboarder

Use for:

- human storyboard workflow expectations
- shot-by-shot review culture
- Fountain integration expectations

Role in this skill:

- what a practical storyboard output should look like
- review-oriented clarity standards

URL:

- https://github.com/wonderunit/storyboarder

### screenplay-parser and screenplay interchange tools

Use for:

- screenplay normalization
- scene, character, and dialogue extraction

Role in this skill:

- deterministic parsing layer before storyboard logic

URLs:

- https://github.com/alexc-hollywood/screenplay-parser
- https://github.com/alexc-hollywood/screen-json

## What not to claim

Do not say this skill is an official release from any of the teams above.

Say instead:

- this skill is based on workflow ideas from these open-source projects
- this skill adapts those ideas into a local storyboard workflow
