# Continuity Rules

The skill should preserve story continuity across prompts.

## Identity anchors

For each important character, keep a stable set of anchors:

- age range or silhouette
- hair
- key wardrobe pieces
- distinctive prop
- body type or movement quality

Do not restate all anchors in every shot. Carry the minimum set needed to avoid drift.

## Environment anchors

Track:

- location geometry
- time of day
- weather
- dominant palette
- repeated objects
- state changes caused by the plot

## Shot-to-shot logic

Adjacent shots should not randomly change:

- direction of movement
- screen geography
- light source direction
- weather state
- wardrobe condition
- prop possession
- ambience bed
- musical motif or energy level
- magical sound signature

## Prompt economy

If a detail is already locked by reference assets or previous validated shots, avoid bloating later prompts with the same text unless drift is likely.

## Continuity checklist

Before finalizing prompts, check:

- Does the character still look like the same person?
- Does the environment still look like the same place?
- Does the style drift away from the chosen mode?
- Does the camera grammar feel intentional from shot to shot?
- Does the sound layer still feel like the same world?
- Is any shot over-specified compared with its narrative value?
