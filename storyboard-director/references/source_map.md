# Source Map

## Purpose

Use this file when you need to justify where a storyboard rule came from.

## Storyboard planning

- ViMax
  Use for shot-level storyboard orchestration and consistency logic.
  https://github.com/HKUDS/ViMax

- MovieAgent
  Use for hierarchical scene-to-shot planning and camera structuring.
  https://github.com/showlab/MovieAgent

- Story2Board
  Use for panel-level prompt shaping and grounded storyboard prompt wording.
  https://github.com/DavidDinkevich/Story2Board

## Review and correction

- FilmAgent
  Use for critique-correct-verify logic and role decomposition.
  https://github.com/HITsz-TMG/FilmAgent

- Storyboarder
  Use for practical review expectations and storyboard clarity.
  https://github.com/wonderunit/storyboarder

## Parsing and script structure

- screenplay-parser
  Use for screenplay normalization.
  https://github.com/alexc-hollywood/screenplay-parser

- screen-json
  Use for screenplay interchange thinking and normalized structure.
  https://github.com/alexc-hollywood/screen-json

## Timing, staging, and camera craft

- ASC Shot Craft: Where Do You Put the Camera?
  Use for blocking-first camera placement.
  https://theasc.com/articles/shot-craft-where-do-you-put-the-camera

- ASC Shot Craft: Tools for Camera Movement
  Use for motivated, readable movement.
  https://theasc.com/articles/shot-craft-camera-movement

- Animation Mentor Eric Larson notes
  Use for phrasing action and leading body action clearly.
  https://www.animationmentor.com/wp-content/uploads/2017/06/ELEnt10_Posing.pdf

- Animation Mentor timing tips
  Use for anticipation, timing, spacing, and avoiding unreadable simultaneous action.
  https://content.animationmentor.com/pdfs/TipsAndTricks_Volume1.pdf

## Lighting and materials

- ARRI Lighting Handbook
  Use for key/fill/back relationships and motivated light.
  https://www.arri.com/en/learn-help/lighting/lighting-handbook

- American Cinematographer on lighting a set
  Use for contrast thinking and readability vs drama.
  https://theasc.com/articles/lighting-a-set

- Pixar RenderMan fundamentals lighting
  Use for CG and animated-world storytelling light logic.
  https://renderman.pixar.com/fundamentals-lighting
