# Expression Design

## Purpose

Use this file when the story beat depends on character feeling, reaction, hesitation, or resolve.

Do not assume body motion alone will carry the scene.

## Core rule

If the audience must understand what the character feels, specify the expression chain.

Record:

- expression before action
- expression during action
- expression after result
- expression shift

## Useful expression layers

Prefer concrete visible behavior:

- eye direction
- eyelid openness
- brow tension or lift
- mouth shape
- head tilt
- shoulder line
- chest or body weight shift
- hand, paw, wing, or tail gesture when relevant

Avoid vague labels like:

- sad
- cute
- worried

Instead describe how that is visible.

## Example

Weak:

- character is worried

Better:

- eyes slightly widened, mouth lightly pressed closed, body leaning toward the object, movement cautious

## Handoff beats

When one character fails and another prepares to try:

- first character should show failure or appeal
- second character should show recognition and decision

Do not cut directly from one failed action to the second character already moving unless the scene intentionally wants abruptness.

## Expression validation

Before finalizing a shot, ask:

- can the audience read what the character feels before the action
- can the audience read whether the result changed the feeling
- is the expression shift stronger than the camera move

If no, revise the shot.
